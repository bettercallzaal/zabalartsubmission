# Archive

This folder contains documentation and database migration files that have been applied or are no longer actively needed.

## Contents

### `/docs/`
- Historical documentation, changelogs, and one-time fix summaries
- Release notes and testing checklists
- Security audit reports

### `/database/`
- Applied database migration scripts
- Historical RPC fixes
- Schema updates that have been deployed

## Note
These files are kept for historical reference and audit purposes. Do not delete unless you're certain they're no longer needed for compliance or debugging.
